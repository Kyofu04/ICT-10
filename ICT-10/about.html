About

Blade of the Brave, is a 2d platformer game.
A 2d platformer is essentially a game that has
2d graphics 2d effects and it gives arcade game
nostalgia.

The main game was coded using pygame with python.
The game took about 6 months with the code and another
6 months for the art. The python code was not originally all mine,
some of the code was from python communities in reddit.
